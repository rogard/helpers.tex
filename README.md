# helpers.tex

Reusable [LaTeX3](https://www.latex-project.org/latex3/) modules.

## Usage
```latex
  \usepackage{/path/to/sty/module.sty}
```

## Module
### [boilerplate](https://github.com/rogard/helpers.tex/blob/main/src/erw-boilerplate.sty)

Reusable `expl3`-like code

### [biblatex](https://github.com/rogard/helpers.tex/blob/main/src/erw-biblatex.sty)

In-text citation template exposing a keyval interface.

### [counter](https://github.com/rogard/helpers.tex/blob/main/src/erw-counter.sty)

Counter commands allowing cross-referencing.

### [lang](https://github.com/rogard/helpers.tex/blob/main/src/erw-lang.sty)

Package-agnostic interface for detecting loaded languages.

### [label-content](https://github.com/rogard/helpers.tex/blob/main/src/erw-label-content.sty)

An `xtemplate` to format a label-content combo, such as inside `\addcontentsline`.

### [calendar](https://github.com/rogard/helpers.tex/blob/main/src/erw-calendar.sty)

An `xtemplate` to  map iso dates to quarter, month, week, in long or short format.

### [punctuation](https://github.com/rogard/helpers.tex/blob/main/src/erw-punctuation.sty)

Think `\Incidental{<clause>}`
