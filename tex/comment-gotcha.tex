% -----------------------------------------------------------------
% Gotcha:
%   "\file_input:n breaks expl3 syntax mode"
% Condition:
%   file.tex ends with `\ExplSyntaxOff`
% Usage:
%   ```
%   \ExplSyntaxOn
%   \file_input:n{/path/to/file.tex} 
%   [ more code ]
%   \ExplSyntaxOff
%   ```
% Solution:
%   ```
%   \ExplSyntaxOn
%   \file_input:n{/path/to/file.tex}
%   \ExplSyntaxOn
%   [ more code ]
%   \ExplSyntaxOff
%   ```
